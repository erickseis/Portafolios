Pokedex

Construida con las tecnologias: React, Redux, React Router, javaScript, HTML, CSS, ocupando la POKEAPI para desarrollarla, nos muestra una guia de stats de diversos Pokemon. Tecnologias utilizadas React, Redux, React Router, HTML y CSS.

Crud

Proyecto donde ocupe herramientas de React con Axios mientras se consume una API con los metodos Get, Post, Delete. Tecnologias utilizadas React, HTML y CSS.

App Clima

Construida para mostrar los datos del tiempo en tu localizacion ocupa una API que permite interactuar con estos datos. Tecnologias utilizadas React, HTML y CSS.

Pokedex Basic

Una version mas sencilla de la pokedex con un diseño amigable e intuitivo, muestra las caracteristicas de cada pokemon. Tecnologias utilizadas React, HTML y CSS.

Rick and Morty

Concebida a partir del consumo de una API, ostenta paginacion para separar la cantidad de personajes a mostrar en pantalla, asi como sus caracteristicas, cuenta con una barra para busqueda personalizada por id de cada planeta. Tecnologias utilizadas React, Axios, HTML y CSS.

Ecommerce JavaScript

Frases Celebres

Proyecto realizado a partir de un archivo JSON, donde nos comparte frases celebres de algunos visionarios del pasado, al cambiar la frase se le asigna un nuevo color al estilo mostrado en pantalla, React, HTML y CSS
